# Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript**. This project showcases my skills, projects, education, and contact information in a clean and professional layout.

## Project Overview

The purpose of this portfolio is to present my work, technical skills, and projects in a visually appealing way. It is fully responsive and works across desktop, tablet, and mobile devices.

## Features

-  Attractive Hero Section
-  About Me Section
-  Skills Section
-  Projects Showcase
-  Contact Section
-  Fully Responsive Design
-  Modern UI/UX
-  Smooth Navigation
-  Cross-Browser Compatible

## Built With

- HTML5
- CSS3
- JavaScript
